# File
socket project provip
    client.py          		    : client
    server.py           	    : server
    Misc			    : folder that contains functions and databases

# Instructions:
- Run server.py and client.py to start the program.
- In client.py's EnterIP window, you will be required to type in the HOST (IP address) and PORT, which is 127.0.0.1 and 55000 correspondingly by default.
- After that, in the Sign In window, type in your account's information then click "Log In" or click "Sign Up" to create a new account.
- Type in date, bank and currency to get data, which will be presented in the chat box.
- Click Disconnect to end communication with the server.
